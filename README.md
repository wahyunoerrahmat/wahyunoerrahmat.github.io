## 🌐 Cara Deploy ke GitHub Pages (`username.github.io`)

1. Buat repositori baru di GitHub dengan nama persis sesuai username GitHub Anda ditambah `.github.io`  
   *(Contoh: `wahyunoerrahmat.github.io`)*.
2. Inisialisasi git dan unggah folder ini ke repositori tersebut:
   ```bash
   git init
   git add .
   git commit -m "Initial commit: Modern Portfolio Website"
   git branch -M main
   git remote add origin https://github.com/wahyu2810/wahyu.github.io.git
   git push -u origin main
   ```
3. Buka halaman repositori di GitHub ➔ **Settings** ➔ **Pages**.
4. Pastikan *Branch* diatur ke `main` dan folder ke `/ (root)`, lalu klik **Save**.
5. Website Anda akan online dan dapat diakses di `https://wahyu2810.github.io/wahyu.github.io/`.
